# Patatap-clone
User can make sweet sweet music by pressing keys (+cool effects)

## Stack:
Front end: Howler.Js, Paper.Js

## How is it made?
- Sound effects through Howler.Js library
- Animations on HTML canvas with Paper.Js loops
